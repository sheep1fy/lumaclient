Luma Client

Feature-rich mod for Minecraft Bedrock on Linux

v2.0.0 • MC 26.0

Experience Minecraft Bedrock with powerful HUD modules, performance monitoring, and customization tools. Built for mcpelauncher-client with version-independent OpenGL hooking that works across updates.

Features

📊

FPS Counter

Real-time frame rate monitoring to track your game's performance with precision.

📍

Coordinates Display

Always know your exact position with XYZ coordinate tracking.

🖱️

CPS Counter

Monitor clicks per second for competitive gameplay and speedrunning.

⌨️

Keystrokes

Visual display of WASD and mouse button inputs in real-time.

🔍

Zoom Module

Adjustable FOV for enhanced visibility when you need a closer look.

⚡

Auto Sprint

Automatic sprinting toggle for seamless movement.

📡

Ping Display

Monitor server latency and stay informed about your connection quality.

🎨

HUD Editor

Drag-and-drop interface to position modules exactly where you want them.

🎮

ImGui Interface

Modern GUI system with clean styling. Toggle with the 'K' key.

Installation

1

Prerequisites

Install required development libraries:
sudo apt-get install libx11-dev
2

Build from Source

Clone and compile the project:
git clone https://github.com/sheep1fy/Luma-Client
cd Luma-Client && mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
3

Install to mcpelauncher

Copy the library to your mods directory:
mkdir -p ~/.local/share/mcpelauncher/mods/Luma\ Client/v2.0.0/x86_64/
cp build/libluma_client.so ~/.local/share/mcpelauncher/mods/Luma\ Client/v2.0.0/x86_64/
4

Launch & Configure

Start Minecraft and open the Luma menu:
mcpelauncher-client
Press K in-game to toggle the menu and configure modules.
