QUICK INSTALLATION

1

Install Dependencies

Make sure you have mcpelauncher-client installed on your Linux system.

BASH sudo apt install mcpelauncher-client
2

Download Luma Client

Get the latest release from our GitHub repository or build from source.

BASH git clone https://github.com/sheep1fy/luma-client.git
cd luma-client
3

Build the Project

Compile the mod using CMake with your preferred configuration.

BASH mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
4

Install the Mod

Copy the compiled library to your mcpelauncher mods directory.

BASH cp libluma_client.so ~/.local/share/mcpelauncher/mods/
5

Launch Minecraft

Start Minecraft Bedrock and press the configured key (default: K) to open the Luma Client menu.

BASH mcpelauncher-client
