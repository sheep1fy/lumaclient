Installation

Get started in just a few steps

1

Install mcpelauncher

Make sure you have mcpelauncher installed on your Linux system.

https://github.com/minecraft-linux/mcpelauncher-manifest
2

Download Luma Client

Get the latest release from GitHub or build from source.

wget https://github.com/sheep1fy/Luma-Client/releases/latest/download/luma_client.so
3

Install the Mod

Copy the .so file to your mcpelauncher mods directory.

mkdir -p ~/.local/share/mcpelauncher/mods
cp luma_client.so ~/.local/share/mcpelauncher/mods/
4

Launch & Enjoy!

Start Minecraft and press Right Shift + F3 to open Luma Client.

mcpelauncher-client
