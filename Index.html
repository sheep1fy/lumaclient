Features

Everything you need, nothing you don't

FPS Counter

Monitor your framerate in real-time with a clean, unobtrusive display.

Coordinates

Always know your position with the built-in coordinate display.

CPS Tracker

Track your clicks per second - useful for PvP practice.

Ping Display

Keep an eye on your connection latency while playing.

Keystrokes

Show WASD inputs on screen - great for content creators.

AutoSprint

Automatically sprint when moving forward to save your pinky finger.

Zoom

Smooth camera zoom with adjustable FOV and keybind support.

HUD Editor

Drag and drop to position HUD elements exactly where you want them.

Config System

Save and load your settings - everything persists between sessions.

Installation

Get up and running in a few minutes

Step 1

Install mcpelauncher

You'll need mcpelauncher to run Bedrock Edition on Linux. If you don't have it yet, check out their GitHub page.

https://github.com/minecraft-linux/mcpelauncher-manifest
Step 2

Download Luma Client

Grab the latest release or build from source if you're feeling adventurous.

wget https://github.com/sheep1fy/Luma-Client/releases/latest/download/luma_client.so
Step 3

Move to mods folder

Create the mods directory if it doesn't exist and copy the file.

mkdir -p ~/.local/share/mcpelauncher/mods
cp luma_client.so ~/.local/share/mcpelauncher/mods/
Step 4

Launch and configure

Start Minecraft and press Right Shift + F3 to open the Luma Client menu. Enable the features you want and position the HUD elements.

mcpelauncher-client
